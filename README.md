# CompTIA A+ Quest — V15 (220-1201 + 220-1202)

A gamified, single-file study tool for the **current CompTIA A+ certification (V15)** — Core 1 (220-1201) and Core 2 (220-1202). Eight weeks of structured daily content, XP, levels, streaks, achievements, and 10 embedded mini-quizzes — all running locally in your browser with no signup, no accounts, no paywalls.

> **Why V15?** CompTIA released the 220-1201 and 220-1202 exams on March 25, 2025. The previous 220-1101/1102 exams retired on September 25, 2025 and can no longer be scheduled. This study tool is fully calibrated to the official **V15 exam objectives v4.0**.

## Setup

1. Download `comptia_a_plus_quest.html`.
2. Double-click it (or right-click → Open With → your browser).
3. Optionally drag it into a browser tab to bookmark.

That's it. Everything runs locally. Progress is saved to your browser's `localStorage`, so as long as you use the same browser profile and don't clear site data, your XP, streak, and completions persist.

## What's inside

- **56 days** of structured study content (8 weeks × 7 days), each ~1–2 hours
- **332 tasks** worth **7,890 XP** total (you only need 4,000 to hit max level — there's headroom for skips)
- **10 mini-quizzes** with 50 scenario-style multiple-choice questions and explanations
- **8 levels** from Technician Trainee → A+ Certified Ready
- **8 achievements** (First Boot, Hardware Hero, Network Ninja, Security Sentinel, Grinder, Halfway There, All Done, Focused)

- **Exam date pickers** for Core 1 and Core 2 with countdowns and pacing recommendations
- Every task references its **official CompTIA exam objective** (e.g., `(Obj 3.1)`)
- Reset button if you want to start over

## What changed in V15 (compared to the retired 220-1101/1102)

The new exams are about **87% the same content** as the retired versions, but with the following meaningful changes baked into this study plan:

**Domain weight shifts (Core 1):** Networking up (20% → 23%), Mobile Devices down (15% → 13%), Troubleshooting down (29% → 28%).

**New / increased coverage:**
- AI considerations (privacy, hallucinations, deepfakes, AI in defense and offense)
- Zero Trust security principles
- Windows 11 specifics (TPM 2.0, UEFI, Secure Boot, modern Settings app)
- Newer filesystems: ReFS, XFS, APFS
- Wi-Fi 6/6E and 6 GHz frequency
- Mini-LED displays
- USB-C and Thunderbolt 4 emphasis
- WPA3 and SAE
- Cloud-based productivity tools (Microsoft 365, Google Workspace, iCloud)
- Modern Linux client/desktop topics
- Passkeys and FIDO2 authentication

**Removed / reduced:**
- 2G cellular, GSM/CDMA distinction, PRL updates (gone)
- SCSI, IDE, eSATA emphasis (reduced — focus shifted to NVMe)
- Formal 6-step troubleshooting methodology (no longer testable, but still best practice)
- Older Windows editions

## How tasks reference the exam

Every task ends with an objective code, like `(Obj 1.3)` or `(Obj 5.7)`. These map to numbered objectives in CompTIA's official Exam Objectives PDF (free download). For example, `Obj 1.3` in Core 1 refers to "Given a scenario, configure basic mobile device network connectivity and provide application support."

This means you can cross-reference any study material — Sybex, Mike Meyers, Pearson, ExamCompass — using the same objective codes. They're the universal index for the exam.

## How to use it (the actual study workflow)

1. **Set your exam dates** at the top so the pacing note shows you a recommended day target.
2. **Start at Day 1**, expand the card, and work through the tasks in order. Tap a task to mark it complete and bank XP.
3. **Use the Review Loop prompt** — it's the active-recall step. Don't skip it. Knowledge that you can pull from memory is the only kind that survives the exam room.
4. **Click resource links** in each day to watch the relevant Professor Messer videos or take the linked practice questions.
5. **Take the mini-quiz** when you reach a quiz day. Don't peek at explanations until after you answer. Score 4/5+ to bank +25 bonus XP.
6. **Build your streak.** Doing one task every day is better than five tasks twice a week. The streak bonus exists for a reason.
7. **Don't skip the mock exam days** (Day 14, 28, 42, 53, 54, 55). They're calibration. If you bomb one, slow down and re-do the relevant week before moving on.

## FAQ

**Can I skip days?** You can, but the order is deliberate. Each week builds on the previous. If you have prior IT experience, breeze through familiar tasks and spend the saved time on tougher domains.

**What if I miss a day and break my streak?** It happens. Streak resets to 1 the next time you complete a task. The streak is a motivator, not a moral judgment.

**Can I export my progress?** Not built in. Open your browser's developer tools → Application → Local Storage to view the raw JSON if you really need to back it up.

**What if a resource link breaks?** External sites occasionally restructure. Fall back to Professor Messer's master playlists or browse `examcompass.com` and `crucialexams.com` to find the V15 sections directly.

**Is this affiliated with CompTIA?** No. CompTIA® and A+® are trademarks of CompTIA, Inc. This is an independent study aid built around their published V15 exam objectives.

**Will this guarantee I pass?** No study tool can. But if you complete every task, hit every review loop, score 80%+ on the mocks, and supplement with hands-on practice (build a PC, set up a home network, install Windows 11 from scratch, configure a Linux VM), you'll walk in prepared.

## Tips for success

- **Pair with hands-on lab work.** Pull a laptop apart and put it back together. Build a VM in VirtualBox. Set up a Linux distro. Configure a SOHO router with WPA3. The exam loves "you walk up to a workstation and X happens" scenarios — labs build that intuition.
- **Memorize port numbers cold.** 20/21, 22, 23, 25, 53, 67/68, 80, 110, 137-139, 143, 161/162, 389, 443, 445, 3389. Make flashcards.
- **Practice PBQs separately.** ExamCompass has them. PBQs trip people up because they're scenario-heavy and longer per question — pace yourself.
- **The malware removal 7-step is a guaranteed test target.** Memorize the order cold: Investigate → Quarantine → Disable Restore → Remediate → Schedule scans/updates → Re-enable Restore + create restore point → Educate user.
- **On exam day:** flag uncertain questions, finish the easy ones first, come back. Don't burn time on a single PBQ.

Good luck. You've got this.
